# SomaStreak 

> A gamified social reading platform that turns solo study sessions into a competitive, addictive experience.

**Live Demo:** [somastreak.vercel.app](https://somastreak.vercel.app)

---

## The Problem

Your "To-Be-Read" pile is growing. Content fatigue makes that 140-page technical manual look like a mountain. Reading is usually a solo sport — and that's exactly why most people don't finish what they start.

SomaStreak fixes that.

---

## What It Does

SomaStreak is a gamified reading platform built around the psychology of streaks, social accountability, and competitive progress tracking.

### Core Features

** Live Race Track Dashboard**
Users and their friends are represented by avatars on a live progress dashboard. As you read, your avatar moves. You're not just reading for yourself — you're staying ahead of the pack.

** Streak System**
Borrowing the best parts of Duolingo and Snapchat — visualising your reading consistency turns "I should read" into "I can't break the chain." Streaks are tracked daily and displayed prominently.

**Spoiler-Shielded Reaction Wall**
Drop thoughts, memes, or reactions tied to specific chapters. The catch: your friends only see your reaction once they've reached the same page. No spoilers, full connection.

** SomaStreak Wrapped**
A monthly data-driven breakdown of your reading habits — pages read, peak focus times, streaks hit — turned into a shareable achievement card.

** Neo-Brutalist UI**
Bold lines, vibrant contrasts, and a Pastel-Coquette aesthetic that makes the reading environment feel like somewhere you actually want to be.

## Getting Started

```bash
# Clone the repo
git clone https://github.com/tullahcodes/somastreak.git
cd somastreak

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
